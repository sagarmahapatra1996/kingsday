# kingsday
A Collection of memories presented as an Angular Website to my bestfriend on his birthday

Live Website :
https://kingsdayjun25.firebaseapp.com/#/

Steps to Run the application:
- Clone the Project
- Run npm install
- Run ng serve --open (It will run on default port 4200)
  - To open in other specific port, Run ng serve --port port_number --open
